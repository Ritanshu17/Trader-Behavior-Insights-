# Trader-Behavior-Insights



# Project Title

Bitcoin Market Sentiment vs Trader Performance Analysis

# Overview

This project analyzes the relationship between Bitcoin market sentiment and trader profitability using historical trading data from Hyperliquid.

The goal is to understand how market psychology affects trading outcomes and identify patterns that may inform better trading strategies.

Dataset Used

1️⃣ Bitcoin Market Sentiment Dataset
Columns:

date
classification
value

2️⃣ Hyperliquid Historical Trader Data

Key Columns:

Account
Execution Price
Size USD
Side
Timestamp
Closed PnL

# Methodology

Data Cleaning

Timestamp alignment

Dataset merging

Trader performance analysis

Sentiment correlation

Visualization of insights

# Key Insights

• Trader profitability varies significantly with market sentiment
• Fear markets trigger higher trading frequency
• Some traders outperform during bearish sentiment
